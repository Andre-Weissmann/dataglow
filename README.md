# DATAGLOW

> **Your data, glowing.** A personal data cleaning and analysis tool.

---

## ✨ What is DATAGLOW?

DATAGLOW is a personal tool designed to take messy, raw data and transform it into something clean, clear, and ready to use. Built by a data analyst, for anyone who works with data.

---

## 🚀 Features

- 🧹 **Data Cleaning** — Remove duplicates, handle nulls, fix formatting issues automatically
- 📊 **Data Analysis** — Surface insights quickly without writing complex queries from scratch
- 💡 **Clear Output** — Results that glow — clean, readable, and actionable
- 🔄 **Personal Use** — Built for your own workflows, your own datasets, your own pace
